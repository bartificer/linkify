LinkData

link-data. LinkData

The information that can be used to render a link.

Instances of this class are created from the information extracted from web pages by data transformers.

new LinkData(url, textopt, descriptionopt, extraFieldsopt)

Name Type Attributes Default Description
url string

The link's URL.
text string <optional>

The link's text, defaults to the URL. The value passed is coerced to a string with String().
description string <optional>

A description for the link, defaults to the link text. The value passed is coerced to a string with String().

 {}

An optional set of extra fields to be made available for use in templates under the extraFields key. All values are coerced to strings with String().
